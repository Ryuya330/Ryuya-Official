# ロゴ使用について

## note ロゴ
- ロゴデータは note 公式サイトの「のろご」からダウンロードが必要です
- 配置場所: `assets/images/logos/note-logo.svg`
- 使用制限:
  - 改変禁止（色や形の変更不可）
  - note公式が提供するロゴデータのみ使用可能
  - 商用利用の場合は事前許可が必要

## 手順
1. note公式サイトから「のろご」をダウンロード
2. `note-logo.svg` として保存
3. ガイドラインに従って適切なサイズで使用

## 参考リンク
- note ロゴキット「のろご」: https://note.com/guide/logo
- ブランドガイドライン: https://note.com/guide

--

注意: このフォルダに同梱した `note-app-icon.svg` と `tales-app-icon.svg` は、公式アプリの「見た目」を参考にして作成したサイト用のモック（代替アイコン）です。以下を必ず留意してください。

- これらは公式ロゴではありません。正式なロゴ・アイコンは note 側の配布物（のろご等）をダウンロードして置き換えてください。
- 商用利用やブランド利用については note の使用ガイドラインに従い、必要に応じて事前許諾を取得してください。
- 置き換え手順:
  1. 公式SVGをダウンロード
  2. `assets/images/logos/note-app-icon.svg` を上書き（同名で置換）
  3. `assets/images/logos/tales-app-icon.svg` を上書き（同名で置換）

これらのSVGはWeb向けに最適化したモックです。サイト上の見た目確認や一時運用にのみ使用してください。
